The DevOps Toolbox

12 essential tools across containerisation, CI/CD, IaC, version control, monitoring, and cloud — used by the world's top engineering teams.

Container & Orchestration

Docker
Container

Docker

Docker introduced the concept of packaging applications and their dependencies into lightweight, portable containers. A container includes everything needed to run: code, runtime, libraries, and configuration — running identically across development, staging, and production.

  • Dockerfile declarative image builds
  • Docker Compose for multi-container apps
  • Docker Hub registry with millions of images
  • Layer caching for fast, incremental builds
13M+ Developers
7M+ Images on Hub
2013 Released
K8s
Orchestration

Kubernetes

Originally developed at Google and open-sourced in 2014, Kubernetes has become the de facto standard for container orchestration. It automates deployment, scaling, self-healing, and load balancing across clusters of machines.

  • Declarative configuration via YAML manifests
  • Horizontal pod autoscaling (HPA)
  • Rolling updates and rollbacks with zero downtime
  • Helm charts for templated application packaging
96% Market adoption
5.8K+ Contributors
CNCF Graduated
Helm
Package Mgr

Helm

The package manager for Kubernetes. Helm uses Charts to define, install, and upgrade complex Kubernetes applications. Think of it as apt or brew but for Kubernetes manifests.

  • Reusable chart templates with values injection
  • Release management and rollback support
  • Artifact Hub with thousands of community charts

CI/CD Pipelines

Jenkins
CI/CD

Jenkins

The most widely deployed open-source automation server. Jenkins orchestrates the build, test, and deploy pipeline using a Jenkinsfile (declarative or scripted pipeline), with 1,800+ community plugins.

  • Pipeline-as-Code via Jenkinsfile (Groovy DSL)
  • Distributed builds across agent nodes
  • Massive plugin ecosystem for any integration
  • Blue Ocean modern UI for pipeline visualisation
1,800+ Plugins
300K+ Installations
GH Act
CI/CD

GitHub Actions

Native CI/CD built into GitHub. Workflows are defined in .github/workflows/*.yml and triggered by push, pull request, schedule, or any GitHub event — with thousands of community actions in the Marketplace.

  • Matrix builds for cross-platform testing
  • Reusable workflows and composite actions
  • GitHub-hosted and self-hosted runners
  • OIDC-based cloud authentication
GitLab
CI/CD

GitLab CI/CD

GitLab offers a complete DevOps platform — from source control to monitoring — with built-in CI/CD defined in a .gitlab-ci.yml file. Particularly strong for self-hosted enterprise pipelines.

  • Multi-stage pipeline with DAG support
  • Auto DevOps for zero-config pipelines
  • Built-in container registry and package registry

Infrastructure as Code

Terraform
IaC

Terraform

HashiCorp's Terraform lets you define cloud infrastructure using the HCL declarative language. A single configuration can provision resources across AWS, Azure, GCP, and hundreds of other providers via the Terraform Registry.

  • Plan → Apply workflow with change previews
  • State management with remote backends (S3, Terraform Cloud)
  • Modular, reusable infrastructure components
  • 1,000+ providers in the Terraform Registry
1,000+ Providers
OpenTF Fork: OpenTofu
Ansible
Config Mgmt

Ansible

Red Hat's Ansible is an agentless configuration management tool. Using Playbooks written in YAML, Ansible can configure systems, deploy software, and orchestrate complex multi-tier deployments — no agent installation required.

  • Agentless — uses SSH or WinRM
  • Idempotent playbooks ensure desired state
  • Ansible Galaxy — community role library
  • AWX / AAP for enterprise automation at scale

Version Control

Git
VCS

Git

The distributed version control system created by Linus Torvalds in 2005. Git enables branching strategies like Git Flow, trunk-based development, and GitHub Flow — the foundation of every modern DevOps pipeline.

  • Branching and merging workflows
  • Signed commits and tags
  • Hooks for pre-commit automation
GH
Platform

GitHub

The world's largest code hosting platform with 100M+ developers. GitHub extends Git with pull requests, code review, issue tracking, project boards, Actions CI/CD, and GitHub Copilot AI assistance.

  • Pull request review workflows
  • Branch protection rules
  • GHAS security scanning
GL
Platform

GitLab

A complete DevOps platform in a single application. GitLab offers source control, CI/CD, security scanning, package registry, and monitoring in one self-hostable or SaaS solution — ideal for regulated environments.

  • Single-application DevOps
  • Self-hosted EE option
  • Compliance pipelines

Monitoring & Observability

Prom
Metrics

Prometheus

The leading open-source monitoring system and time-series database. Prometheus scrapes metrics via HTTP, stores them with a powerful query language (PromQL), and fires alerts via Alertmanager.

  • Pull-based metric scraping
  • PromQL query language
  • Kubernetes-native service discovery
Grafana
Dashboards

Grafana

The open-source observability and data visualisation platform. Grafana connects to dozens of data sources — Prometheus, Loki, Tempo, Elasticsearch — and renders beautiful, actionable dashboards.

  • Multi-data-source dashboards
  • Alerting and on-call routing
  • Grafana LGTM observability stack
HashiCorp Vault
Security

HashiCorp Vault

Centralised secrets management and data protection. Vault stores API keys, passwords, certificates, and tokens — with dynamic secret generation, lease renewal, and revocation for zero long-lived credentials.

  • Dynamic secrets with automatic rotation
  • PKI and certificate authority
  • Fine-grained policies via HCL
