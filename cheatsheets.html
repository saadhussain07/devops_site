Docker

Container lifecycle, images, networking, volumes

Build imageessential
docker build -t myapp:1.0 .
Build with no cache
docker build --no-cache -t myapp:1.0 .
Run containeressential
docker run -d -p 8080:80 --name myapp myapp:1.0
Run interactive shell
docker run -it myapp:1.0 /bin/sh
Exec into running container
docker exec -it myapp /bin/bash
View logs (follow)
docker logs myapp -f --tail 100
List running containers
docker ps
List all containers
docker ps -a
Stop container
docker stop myapp
Remove container
docker rm myapp
List images
docker images
Remove image
docker rmi myapp:1.0
Push to registry
docker push myuser/myapp:1.0
Pull image
docker pull nginx:alpine
System cleanupdestructive
docker system prune -af --volumes
Inspect container
docker inspect myapp
Copy file from container
docker cp myapp:/app/log.txt ./log.txt
Stats (live resource usage)
docker stats
Docker Compose up
docker compose up -d --build
Docker Compose down
docker compose down -v

Kubernetes (kubectl)

Cluster management, deployments, debugging

Get all pods
kubectl get pods -A
Get pods with details
kubectl get pods -o wide
Describe pod
kubectl describe pod <pod-name>
View pod logs
kubectl logs <pod-name> -f --tail=100
Exec into pod
kubectl exec -it <pod-name> -- /bin/sh
Apply manifest
kubectl apply -f deployment.yaml
Delete resource
kubectl delete -f deployment.yaml
Scale deployment
kubectl scale deployment myapp --replicas=5
Rolling update image
kubectl set image deployment/myapp app=myapp:2.0
Rollback deployment
kubectl rollout undo deployment/myapp
Rollout status
kubectl rollout status deployment/myapp
Get services
kubectl get svc -A
Port forward
kubectl port-forward svc/myapp 8080:80
Get namespaces
kubectl get ns
Set namespace context
kubectl config set-context --current --namespace=prod
Get nodes
kubectl get nodes -o wide
Top pods (resource usage)
kubectl top pods --sort-by=memory
Get events
kubectl get events --sort-by=.lastTimestamp
Apply all in directory
kubectl apply -f ./k8s/
Force delete stuck podforce
kubectl delete pod <name> --grace-period=0 --force

Git

Version control, branching, history, remotes

Clone repo
git clone https://github.com/user/repo.git
Stage all changes
git add -A
Commit with message
git commit -m "feat: add new feature"
Push to remote
git push origin main
Create & switch branch
git checkout -b feature/my-feature
Switch branch
git switch main
Pull latest changes
git pull origin main --rebase
View status
git status
View log (pretty)
git log --oneline --graph --all
Stash changes
git stash push -m "work in progress"
Pop stash
git stash pop
Rebase onto main
git rebase origin/main
Squash last 3 commits
git rebase -i HEAD~3
Undo last commit (keep changes)
git reset --soft HEAD~1
Discard file changes
git checkout -- filename.txt
Create tag
git tag -a v1.0.0 -m "Release 1.0.0"
Push tags
git push origin --tags
List remotes
git remote -v

Terraform

IaC — init, plan, apply, state, workspaces

Initialise projectalways first
terraform init
Preview changes
terraform plan
Apply changes
terraform apply
Apply without prompt
terraform apply -auto-approve
Destroy all resourcesdestructive
terraform destroy
Validate config
terraform validate
Format code
terraform fmt -recursive
Show state
terraform show
List state resources
terraform state list
Remove resource from state
terraform state rm aws_instance.web
Import existing resource
terraform import aws_instance.web i-1234567890
Create workspace
terraform workspace new staging
Switch workspace
terraform workspace select prod
Output values
terraform output
Plan with var file
terraform plan -var-file=prod.tfvars

Linux / Shell

Files, processes, networking, system info

Find files by name
find /var/log -name "*.log" -mtime -7
Search inside files
grep -r "error" /var/log/ --include="*.log"
Disk usage
du -sh /* | sort -rh | head -20
Check disk space
df -h
System resource usage
htop
List open ports
ss -tulnp
Kill process by name
pkill -f "process-name"
Follow log file
tail -f /var/log/syslog
Archive directory
tar -czf archive.tar.gz ./directory/
Extract archive
tar -xzf archive.tar.gz
SCP file to remote
scp file.txt user@host:/remote/path/
SSH with key
ssh -i ~/.ssh/key.pem user@host
Change permissions
chmod 755 script.sh
Change owner
chown -R www-data:www-data /var/www/
Crontab edit
crontab -e
Systemctl service status
systemctl status nginx
View memory usage
free -h
Check network connectivity
curl -I https://example.com

Helm

Kubernetes package manager — install, upgrade, rollback

Add repo
helm repo add stable https://charts.helm.sh/stable
Update repos
helm repo update
Search for chart
helm search repo nginx
Install chart
helm install myrelease stable/nginx
Install with values file
helm install myapp ./chart -f values.prod.yaml
Upgrade release
helm upgrade myapp ./chart -f values.yaml
Upgrade or install
helm upgrade --install myapp ./chart
Rollback release
helm rollback myapp 1
List releases
helm list -A
Uninstall release
helm uninstall myapp
Dry run install
helm install myapp ./chart --dry-run --debug
Show chart values
helm show values stable/nginx
🤖

Ansible

Agentless automation — playbooks, inventory, ad-hoc

Ping all hosts
ansible all -m ping -i inventory.ini
Run playbook
ansible-playbook -i inventory.ini playbook.yml
Dry run (check mode)
ansible-playbook playbook.yml --check
Run with extra vars
ansible-playbook playbook.yml -e "env=prod"
Limit to specific host
ansible-playbook playbook.yml --limit web01
Ad-hoc command on all
ansible all -m shell -a "uptime"
List hosts in inventory
ansible-inventory -i inventory.ini --list
Encrypt file with vault
ansible-vault encrypt secrets.yml
Run with vault password
ansible-playbook playbook.yml --ask-vault-pass
Verbose output
ansible-playbook playbook.yml -vvv
📊

Prometheus (PromQL)

Metrics queries — RED method, resources, alerts

Request rate (RED: Rate)
rate(http_requests_total[5m])
Error rate (RED: Errors)
rate(http_requests_total{status=~"5.."}[5m]) / rate(http_requests_total[5m])
P99 latency (RED: Duration)
histogram_quantile(0.99, rate(http_request_duration_seconds_bucket[5m]))
CPU usage percent
100 - (avg by(instance)(rate(node_cpu_seconds_total{mode="idle"}[5m])) * 100)
Memory usage percent
(1 - node_memory_MemAvailable_bytes / node_memory_MemTotal_bytes) * 100
Disk usage percent
(1 - node_filesystem_free_bytes / node_filesystem_size_bytes) * 100
Pod restart count
kube_pod_container_status_restarts_total
Pods not running
kube_pod_status_phase{phase!="Running"}
Increase over time
increase(http_requests_total[1h])
Top 5 by metric value
topk(5, rate(http_requests_total[5m]))
Sum by label
sum by(service) (rate(http_requests_total[5m]))
Alert: high error rate
rate(http_requests_total{status=~"5.."}[5m]) > 0.05
