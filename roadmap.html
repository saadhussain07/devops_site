How to use this roadmap

Work through phases in order. Don't skip foundations — every advanced DevOps concept builds on Linux, networking, and Git fundamentals. Click any skill item to mark it complete. Your progress saves to your browser automatically.

Phase 1 — Foundations
🐧
Linux & Shell Scripting
The bedrock of all DevOps work
  • Linux filesystem & permissions
  • Shell commands (grep, sed, awk, find)
  • Bash scripting — variables, loops, functions
  • Process management (ps, top, systemctl)
  • Cron jobs & task scheduling
  • SSH, SCP, and secure key management
0/6 Beginner ~3 wks
🌐
Networking Fundamentals
How systems talk to each other
  • OSI model & TCP/IP stack
  • DNS, DHCP, and IP addressing
  • HTTP/HTTPS, TLS/SSL certificates
  • Load balancers & reverse proxies
  • Firewalls, VPNs, security groups
0/5 Beginner ~2 wks
🌿
Git & Version Control
Source of truth for everything
  • Git basics — clone, add, commit, push
  • Branching strategies (Git Flow, trunk-based)
  • Pull requests & code review process
  • Merge vs rebase — when to use each
  • Git hooks for pre-commit automation
  • Semantic versioning & tagging
0/6 Beginner ~2 wks
💻
Programming for DevOps
Automate everything
  • Python scripting — files, APIs, parsing
  • YAML & JSON — read, write, validate
  • Regular expressions (regex)
  • REST API calls with curl & Python requests
  • Makefiles & Taskfiles for automation
0/5 Beginner ~4 wks
Phase 2 — Core DevOps
🐳
Containers & Docker
Package once, run anywhere
  • Docker architecture & container runtime
  • Writing efficient Dockerfiles & .dockerignore
  • Docker layer caching & multi-stage builds
  • Docker Compose for local dev environments
  • Container registries — Docker Hub, ECR, GCR
  • Container security scanning (Trivy, Snyk)
0/6 Intermediate ~3 wks
🔁
CI/CD Pipelines
Automate the path to production
  • GitHub Actions — workflows, triggers, jobs
  • Jenkins — Jenkinsfile, pipelines, agents
  • Artifact management & versioning
  • Pipeline security — secrets, OIDC, signing
  • Test automation integration in pipelines
  • Deployment gates & approval workflows
0/6 Intermediate ~4 wks
🏗️
Infrastructure as Code
Infrastructure in version control
  • Terraform — providers, resources, state
  • Terraform modules & remote backends
  • Ansible — playbooks, roles, inventory
  • Policy as Code — OPA, Sentinel
  • Drift detection & remediation
0/5 Intermediate ~5 wks
☁️
Cloud Platforms
AWS, GCP, or Azure — pick one first
  • Core services — compute, storage, networking
  • IAM — roles, policies, least privilege
  • Managed databases (RDS, Cloud SQL)
  • Serverless — Lambda, Cloud Functions
  • Cost optimisation & reserved capacity
  • Cloud certifications (AWS SAA, GCP ACE)
0/6 Intermediate ~8 wks
📊
Monitoring & Observability
See what's happening inside your systems
  • Prometheus — scraping, PromQL, alerting
  • Grafana — dashboards & alerting rules
  • Log aggregation — ELK stack or Loki
  • Distributed tracing — Jaeger, Tempo
  • SLOs, SLAs, error budgets
  • On-call, runbooks & incident response
0/6 Intermediate ~4 wks
Phase 3 — Advanced
⚙️
Kubernetes Deep Dive
The orchestration standard
  • K8s architecture — control plane, nodes
  • Deployments, StatefulSets, DaemonSets
  • Helm — charts, values, releases
  • RBAC — roles, bindings, service accounts
  • HPA, VPA & cluster autoscaling
  • Network policies & Ingress controllers
  • CKA/CKAD certification
0/7 Advanced ~8 wks
🔀
GitOps & CD at Scale
Git as the single source of truth
  • GitOps principles & the four pillars
  • Argo CD — apps, sync, health checks
  • Flux CD — GitRepository, Kustomize
  • Canary & blue-green via Argo Rollouts
  • Image automation & update policies
0/5 Advanced ~4 wks
🛡️
DevSecOps
Security shifted left
  • SAST — static code analysis in pipelines
  • DAST — dynamic application scanning
  • Secrets management — Vault, AWS SM
  • Container image scanning (Trivy, Grype)
  • Supply chain security — SBOM, Sigstore
  • Zero-trust networking & mTLS
0/6 Advanced ~5 wks
🧩
Microservices & Service Mesh
Build for scale and resilience
  • Microservices patterns — sidecar, ambassador
  • Istio / Linkerd service mesh setup
  • gRPC & Protocol Buffers
  • Event-driven architecture — Kafka, RabbitMQ
  • API gateway patterns & rate limiting
0/5 Advanced ~6 wks
Phase 4 — Expert / Platform Engineering
🚀
Platform Engineering
Build the golden path for developers
  • Internal Developer Platform (IDP) design
  • Backstage.io — service catalog & scaffolding
  • Self-service infrastructure with Crossplane
  • Developer experience (DX) metrics
  • Golden path templates & paved roads
0/5 Expert ~8 wks
📡
Site Reliability Engineering
Run at scale, reliably
  • SRE principles from Google's book
  • Error budgets & SLO-based alerting
  • Chaos engineering — Chaos Monkey, LitmusChaos
  • Capacity planning & load testing
  • Blameless post-mortems & RCA
  • Toil measurement & elimination
0/6 Expert ~6 wks
💰
FinOps & Cloud Cost
Optimise the bill
  • Cloud cost visibility & tagging strategy
  • Reserved instances & savings plans
  • Spot / preemptible instances for workloads
  • Kubecost & OpenCost for K8s cost visibility
  • FinOps Foundation certification
0/5 Expert ~4 wks
🤖
AI-Ops & MLOps
The next frontier of DevOps
  • ML pipeline CI/CD — training & serving
  • Model registries — MLflow, Vertex AI
  • AI-assisted incident response (AIOps)
  • LLM deployment — vLLM, Ollama on K8s
  • GitHub Copilot & AI code review in CI
0/5 Expert ~6 wks
🎉
